hexo+github 搭建个人博客

2017-03-08
Blog

为什么选择使用Hexo在github上搭建博客
博客搭建的三个阶段

  • 第一阶段，刚接触Blog，觉得很新鲜，试着选择一个免费空间来写。
  • 第二阶段，发现免费空间限制太多，就自己购买域名和空间，搭建独立博客。
  • 第三阶段，觉得独立博客的管理太麻烦，最好在保留控制权的前提下，让别人来管，自己只负责写文章。
    第一阶段最简单，在csdn、博客园或者其他博客网站上注册一个账号就算开个blog了。
    第二阶段功能强大，购买空间之后就能创建动态网站，然而还要自己写代码，对于工作党来说，时间宝贵，花费在这个上面也提升不了多少自己的价值。
    而搭建该博客的目的是在于提供一种形式让自己去总结自己的工作经验、技术经验，也当作一个分享。所以，我选择的是第三种方法。

Hexo
Hexo是一个简单、快速、强大的静态博客框架,出自台湾大学生tommy351之手。使用Jekyll搭建个人博客的过程，确实要繁琐许多。相比之下Hexo更轻便更快捷，下面是Hexo官网强调Hexo的四大特点：

  • 极速生成静态页面
  • 支持Markdown
  • 丰富的插件支持
  • 一键部署博客

一、环境准备
1、安装git ： http://pan.baidu.com/s/1eS1fJRs

2、安装node.js ：http://pan.baidu.com/s/1o8He5HS
    * 首先hexo是基于nodejs的，所以必须安装nodejs
    * 安装nodejs方法很多，我选择homebrew安装方式，所以需要安装它
    * 安装homebrew就很简单了，mac自带ruby脚本功能，一句话搞定
    * hexo提交部署github需要使用git工具，所以需要安装git，用homebrew的话也是一句话搞定

二、安装和配置hexo

* npm install -g hexo
* npm install -g hexo-cli
* npm install hexo --save

三、使用Hexo创建博客框架
1、 在D盘新建一个文件夹 hexo，右键打开 Git Bash Here ，执行以下命令

* hexo  init
* npm   install
* hexo  g   --生成博客静态页
* hexo  s   --本地发布博客，访问地址 localhost：4000

四、使用GitHub Pages建立博客站点

GitHub Pages，是用来托管GitHub上静态网页的免费站点，那GitHub Pages具体有哪些功能呢：

    * 有300M免费空间，资料自己管理，保存可靠；
    * 享受 GitHub 的便利，上面有很多大牛，眼界会开阔很多；
    * 可以使用很多现成的博客框架，搭建过程简单快速。
1、注册一个github账号：https://github.com
2、创建一个****.github.io的仓库, ****位置为github账号，此处必须一致！
3、配置git
   （1）、全局配置下邮箱和名字（注册github时的邮箱和账号）,命令如下
         * git config --global user.email "github邮箱"
         * git config --global user.name "github名字"
   （2）、配置SSH
        3-2-1、 执行下面命令生成SSH
          * ssh-keygen      --敲三次回车，既可以生成id_rsa.pub文件，这里面就是SSH key的内容
        3-2-2、打开你的github,点击Setting -> deploy key -> add deploy ,将id_rsa.pub文件内容粘贴进
        去，并勾选allow write access
        3-2-3、判断SSH是否配置好,命令如下：
          * ssh -T git@github.com
          出现Hi CoderTian! You've successfully authenticated, but GitHub does not provide shell access.
          就表示你已经配置好了SSH
4、配置本地Hexo
   （1）、打开Hexo目录下的_config.yml，拉倒最下面
          配置为这样子,只需要把xiaosecond改为你自己的github用户名就可以了。
            * deploy:
            * type: git
            * repository:  git@github.com:xiaosecond/xiaosecond.github.io.git
            * branch: master
   (2)、保存之后回到hexo 根目录，依次执行如下命令：
            * hexo  clean
            * hexo  g
            * hexo  d
         显示以下内容表示上传成功！
            To github.com:xiaosecond/xiaosecond.github.io.git
             + 662555e...23556e2 HEAD -> master (forced update)
            INFO  Deploy done: git
   （3）、  打开你的github,点击Setting -> options ->  GitHub Pages  开启主页功能
         如果成功的话在浏览器输入(http://xiaosecond.github.io) 就可以访问你的博客了。把用户名换为你自己的。
5、绑定域名
   1、ping到 github 上的ip,命令如下;
      * ping  xiaosecond.github.io
   2、登陆阿里云，把获取的IP绑定到域名上，首先找到控制台-->产品与服务-->域名-->域名列表
   3、在xiaosecond.github.io的根目录下添加CNAME,具体就是在Hexo目录里的source文件下添加
      一个名为CNAME的文件，注意这个文件是没有后缀的，千万不要设置成.txt文本文件，文件的
      内容就是域名，格式如：
      * xiaosecond.top
     添加后部署文件，这个时候再试，应该就成功了
